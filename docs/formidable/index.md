# Formidable

Formidable is an ExpressionEngine addon for building forms two ways: capture
existing HTML5 forms straight out of your templates, or build them field by
field in the control panel. Either path gets you the same feature set --
conditional fields, multi-action workflows, and connections to third-party
services -- backed by the same submission and validation pipeline.

## What's here

- **[Installation](./installation.md)** -- requirements and setup.
- **[The Capture tag](./capture-tag.md)** -- turn existing template
