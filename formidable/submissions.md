# Submissions

**Formidable &raquo; Submissions** lists every stored submission across your
forms (only forms with **Store Submissions** enabled write here -- see
[the Form tag](/formidable/form-tag/)).

## List and detail

The list is filterable by form and status, and shows the submission's name
column, form, status, and created date. Opening a submission shows every
field's stored value alongside the submission's member (if the visitor was
logged in) and status.

## Statuses

Each form has its own set of statuses (with core defaults plus any custom
statuses you add), reorderable per form. Changing a submission's status from
the detail screen fires the **Submission Updated** workflow event.

## Export

Use **Export CSV** from the submissions list to export the current
filtered view.
